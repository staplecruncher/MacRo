# MacRo

MacOS bootstrapper & fast flag editor for the Roblox Player and Roblox Studio.

Included is the flag `DFFlagDisableDPIScale: true`, which fixes the display scaling issue that causes Roblox to render in your scaled resolution, instead of native resolution.

![MacRo main window](docs/images/main-window.png)
![MacRo menu bar](docs/images/menu-bar.png)

## Requirements

- MacOS 14 or newer
